# TCP-UDP Programs

A collection of C programs covering TCP and UDP socket programming, written during the **Computer Network Lab** at **KIIT (Kalinga Institute of Industrial Technology)**.

[![C](https://img.shields.io/badge/Language-C-A8B9CC?style=flat-square&logo=c&logoColor=white)](#)
[![KIIT](https://img.shields.io/badge/KIIT-Network_Lab-003366?style=flat-square)](#)

## What's Inside

Programs demonstrating:
- TCP connection establishment and data transfer (connection-oriented, reliable)
- UDP datagram communication (connectionless, low-latency)
- Socket creation, binding, sending, and receiving
- Basic network error handling
